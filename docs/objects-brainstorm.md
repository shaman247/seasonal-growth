# Seasonal Growth - Object Brainstorm

## Current Size Distribution Analysis

The player starts at 12px and needs to grow to ~475px to roll up the Sun (380px). Objects can be collected when player is 125% of object size.

### Current Gaps
- **30-70px (Medium)**: Very few options - need more mid-game content
- **70-120px (Large)**: Almost empty - critical gap in progression
- **120-200px (Very Large)**: Only trees/rocks (obstacles) - need collectibles

1. **Consistent art style** - pixel art or stylized 2D

- AI image generation with consistent style prompts
- Sprite sheet format for efficiency
